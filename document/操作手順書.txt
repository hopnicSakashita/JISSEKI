====================================================================
              生産実績管理システム（JISSEKI）操作手順書
====================================================================
作成日: 2025年6月（最終更新: 2025年6月）

■ 1. システム起動・ログイン
────────────────────────────────────────────────────────────────

【1.1 システムアクセス】
1. Webブラウザを起動
2. システムURL にアクセス
3. ログイン画面が表示される

【1.2 ログイン手順】
1. ユーザーID を入力
2. パスワード を入力
3. 「ログイン」ボタンをクリック
4. 認証成功で実績CSV取込画面に遷移

■ 2. CSVデータ取込操作
────────────────────────────────────────────────────────────────

【2.1 実績CSV取込手順】
1. ナビゲーションメニューから「CSV」→「実績CSV取込」を選択
2. SpeedyNeoからのCSVファイル準備：
   - SpeedyNeo→「工程」→「実績一覧」
   - R1注入日で過去2週間程度を範囲指定
   - 「抽出」→「CSV」でファイル出力
3. 「ファイルを選択」ボタンをクリック
4. 取込対象のCSVファイルを選択（Shift-JIS形式）
5. 「アップロード」ボタンをクリック
6. 処理完了メッセージを確認

【2.2 その他CSV取込】
- 注入日・検査日CSV取込：注入日・検査日情報の更新
- バーコードCSV取込：バーコード情報の登録
- 指示データCSV取込：作業指示情報の登録
- 膜カットデータCSV取込：膜カット作業実績の登録
- 膜加工データCSV取込：膜加工作業実績の登録
- スピンコートデータCSV取込：SPC工程データの登録
- ハードコートデータCSV取込：HDC工程データの登録

【2.3 エラー発生時の対応】
- ファイル形式エラー → CSV形式で再保存
- エンコードエラー → Shift-JISで再保存
- 重複エラー → 既存データとの重複確認

■ 3. 実績・進捗確認操作
────────────────────────────────────────────────────────────────

【3.1 R1・R2実績検索手順】
1. ナビゲーションメニューから「実績」→「R1・R2実績」を選択
2. 検索条件を入力：
   - R1注入日（開始～終了）
   - R2注入日（開始～終了）
   - R1注入者、R2注入者
3. 日付指定ボタンの活用：
   - 「今日」「昨日」「今月」「前月」ボタンで簡単設定
4. 「検索」ボタンをクリック
5. 検索結果をスクロール表示で確認（高さ600px）

【3.2 進捗状況確認手順】
1. ナビゲーションメニューから「実績」→「進捗状況」を選択
2. 検索条件を設定：
   - R1注入日（開始～終了）
   - モノマー種別
3. 「検索」ボタンをクリック
4. モノマー種別ごとの進捗状況を確認：
   - R1注入、R2注入、離型、アニール
   - 一次検査、二次検査、三次検査
5. 完了/全体数と色分け表示で進捗を把握

【3.3 未完了案件確認】
各未完了画面で進捗遅延の管理：
- R1重合未完了：最重要（後工程への影響大）
- R1検品未完了：検品待ち案件の管理
- R2注入未完了：高重要度
- R2重合未完了：R2重合予定と実績管理
- 離型未完了：緊急度順・日付順で表示
- 未検査品一覧：検査待ち品の管理
- アニール未完了一覧：熱処理工程の管理

■ 4. 分析画面の活用
────────────────────────────────────────────────────────────────

【4.1 不良率分析の手順】
1. ナビゲーションメニューから「分析」→「不良率分析」を選択
2. フィルター設定：
   - R1注入日（開始～終了）
   - R2注入日（開始～終了）
   - 検査日（開始～終了）
   - 製品ID、モノマー種、製品名、膜カラー
3. 日付指定ボタンで期間設定（今日、昨日、今月、前月）
4. 「適用」ボタンをクリック
5. グラフ表示で分析結果を確認：
   - グラフタイプ：棒グラフ/レーダーチャート選択可
   - 詳細データテーブルで数値確認

【4.2 モノマー種別毎の良品率分析】
1. ナビゲーションメニューから「分析」→「モノマー種別毎の良品率分析」を選択
2. フィルター設定：
   - R2注入日（開始～終了）
   - 注意：R2注入日CSVを取り込まないと正しく検索できない可能性があります
3. 「適用」ボタンをクリック
4. 分析結果の確認：
   - 期間合計テーブル
   - モノマー種別詳細データテーブル
   - 目標値超過時の赤字強調表示
   - グラフ表示

【4.3 その他分析機能】
- モノマー種別不良詳細：根本原因分析
- 不良率上位項目：重点改善対象の特定
- 不良項目別分析：不良項目の傾向把握
- モノマー種別達成率：生産計画達成状況
- モノマー種別検査実績：品質検査の体系的管理
- 膜カット不良率分析：膜カット工程の品質改善

■ 5. 膜カット関連操作
────────────────────────────────────────────────────────────────

【5.1 膜カットデータの取込】
1. 「膜カットデータCSV取込」画面にアクセス
2. CSVファイルをアップロード
3. データ処理結果を確認

【5.2 膜カット不良率分析】  
1. 「分析」→「膜カット不良率分析」を選択
2. 検索条件を設定（カット日、モノマー等）
3. 検索ボタンをクリック
4. 分析結果とグラフを確認

【5.3 膜カット モノマー別集計】
1. 「分析」→「膜カット モノマー別集計」を選択
2. カット日の範囲を設定
   - 開始日・終了日を入力（必須）
   - または「今月」ボタンで今月の期間を自動設定
3. 「検索」ボタンをクリック
4. 結果確認
   - 上部：全体集計（総投入数、総良品数、総合格数、各種歩留率）
   - 下部：モノマー別詳細表（モノマー別の数量と歩留率）
5. 必要に応じて期間を変更して再検索

【5.4 膜カット合格数クロス集計】
1. 「実績」→「膜カット合格数クロス集計」を選択
2. 検索条件を設定：
   - カット日（開始）：検索開始日を入力
   - カット日（終了）：検索終了日を入力
   - モノマー：プルダウンから選択（「すべて」も選択可能）
3. 「検索」ボタンをクリック
4. クロス集計表の確認：
   - 縦軸（行）：色別データ
   - 横軸（列）：膜カーブ別データ  
   - セル内容：該当条件の合格数
   - 行合計：色別の合計数量
   - 列合計：膜カーブ別の合計数量
   - 総合計：全体の合計数量
5. 「クリア」ボタンで条件をリセット可能

【活用方法】
- 色別・膜カーブ別の生産バランス確認
- 特定期間の生産実績の俯瞰的把握
- モノマー別の生産傾向分析
- 月次・週次報告書作成時のデータ確認

【注意事項】
- 色目データ（モノマー=8、カットメニュー=62）は集計から除外されます
- 歩留率は以下の計算式で算出されます：
  * 良品率 = (良品数 ÷ 投入数) × 100
  * 合格率 = (合格数 ÷ 良品数) × 100  
  * 総合歩留率 = (合格数 ÷ 投入数) × 100

■ 6. スライド・設定操作
────────────────────────────────────────────────────────────────

【6.1 システム設定変更】
1. ナビゲーションメニューから「SLD」→「設定」を選択
2. 設定項目の変更：
   - 描画間隔
   - 抽出期間
   - お知らせ情報（ヘッダ1～3、内容1～3）
3. 「保存」ボタンをクリック

【6.2 スライド表示】
1. ナビゲーションメニューから「SLD」→「モノマー種別スライド」を選択
2. 自動更新スライドショーが開始
3. 現場モニター表示や会議資料投影に活用
4. 全画面表示：F11キー

■ 7. 石田第一・第二工程操作
────────────────────────────────────────────────────────────────

【7.1 石田第一（膜加工関連）】
- 膜加工データCSV取込：膜加工作業実績の登録
- 膜加工データ：膜加工工程のデータ分析
- カラー不良データ入力用：色品質・透明度の管理
- 不良分析（スライド）：色品質状況の見える化

【7.2 石田第二（コート関連）】
- スピンコートデータCSV取込：SPC工程データの登録
- ハードコートデータCSV取込：HDC工程データの登録
- 各種不良率分析：SPC・HDC工程の品質管理
- 種類別分析：製品種別の品質特性把握
- ハードコートスピンコートスライダー：HDC SPCスライダー

■ 8. データ編集・管理
────────────────────────────────────────────────────────────────

【8.1 膜カットデータ新規作成】
1. ナビゲーションメニューから「実績」→「膜カットデータ一覧」を選択
2. 「新規作成」ボタンをクリック
3. 必須項目を入力：
   - カット日（必須）
   - モノマー（必須）
   - アニール番号（必須）
   - カット機番号（必須）
   - アイテム（必須）
   - カットメニュー（必須）
   - 膜加工日（必須）
   - 膜カーブ（必須）
   - 色（必須）
   - 投入数（必須）
4. その他項目を必要に応じて入力
5. 「保存」ボタンをクリック

【8.2 膜カットデータ編集】
1. 膜カットデータ一覧で編集対象を選択
2. 「編集」リンクをクリック
3. 必要項目を修正
4. 「更新」ボタンをクリック

【8.3 カラー不良データ入力】
1. ナビゲーションメニューから「石田第一」→「カラー不良データ入力用」を選択
2. 不良種別、数量を入力
3. 発生原因、対策内容を記録

■ 9. トラブルシューティング
────────────────────────────────────────────────────────────────

【9.1 よくあるエラーと対処法】

◆ ログインできない
- ユーザーID・パスワードの確認
- CapsLockの状態確認
- ブラウザのキャッシュクリア

◆ CSVファイルアップロードエラー
- ファイル形式の確認（CSV形式）
- エンコーディングの確認（Shift-JIS）
- ファイルサイズの確認
- SpeedyNeoからの正しい出力手順の確認

◆ 検索結果が表示されない
- 検索条件の見直し
- データ期間の確認
- 対象CSVの取込状況確認
- 権限の確認

◆ 分析グラフが表示されない
- ブラウザの更新（F5キー）
- JavaScriptの有効化確認
- 対象データの有無確認
- フィルター条件の見直し

◆ 進捗データが正しく表示されない
- 注入日・検査日CSVの取込確認
- R2注入日CSVの取込確認
- データの最新性確認

【9.2 システム管理者への連絡】
以下の場合は管理者へ連絡：
- システムエラーの発生
- データの異常値検出
- 新機能の要望
- 権限変更の必要
- CSV取込処理の不具合

■ 10. 効率的な運用のコツ
────────────────────────────────────────────────────────────────

【10.1 日常業務での活用】
- 朝一番に進捗状況画面で当日の作業確認
- R1・R2実績で前日の生産実績確認
- 未完了案件一覧で遅延案件の優先対応
- 定期的な不良率分析で品質動向把握

【10.2 データ取込の効率化】
- SpeedyNeoからの定期的なCSV出力
- 注入日・検査日CSVの適時取込
- 石田第一・第二での工程別データ取込

【10.3 分析業務での活用】
- 期間を変えた比較分析
- モノマー種別毎の良品率分析で原材料品質管理
- 不良率上位項目で重点改善対象の特定
- 複数の分析画面を組み合わせた多角的分析

【10.4 現場での活用】
- スライド機能での見える化
- 進捗状況のリアルタイム把握
- 未完了案件の優先度管理

【10.5 会議・報告での活用】
- スライド機能での見える化
- 分析結果の詳細データ活用
- リアルタイムデータでの現状報告
- 目標値との比較による改善効果測定
